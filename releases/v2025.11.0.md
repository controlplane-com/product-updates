# Release 2025.11.0

## Platform Updates - November 2025

- **Managed Kubernetes**
  - Added GCP provider support for mk8s clusters
  - New Headlamp optional add-on for enhanced cluster management and visualization
  - Improved API connector with connection management and connection pool limits
  - Updated MetalLB for easier generic cluster external routing

- **MCP Server**
  - Introduced new MCP (Model Context Protocol) server for Control Plane https://mcp.cpln.io
  - Added comprehensive tooling and guidance for cpln cli and manifest generation
  - Enforced health check requirements for deployments

- **Billing**
  - Improved API latency for regular queries and monthly report generation
  - Enhanced billing metrics reporting

- **Workloads**
  - Show error messages per replica for stateful workloads
  - Create gateways on startup for all BYOK orgs
  - Fixed cache invalidation when removing and adding locations on GVC
  - Fixed issue preventing unused shared volume sets from being cleaned up

- **Data Service**
  - Improved cache design for SAML group matching

- **Storage & Object Store**
  - Improved reliability for Azure file storage volume mounts

- **Additional Updates**
  - Improved terraform-exporter to convert multiline set strings to heredocs
  - Domain maxAge setting must now be a valid golang duration
  - Metering endpoint added for monthly tag aggregation across all orgs
