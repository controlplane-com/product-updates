# Release 2026.03.0

## Platform Updates - March 2026

- **Managed Kubernetes**
  - Upgraded Istio to 1.27.8
  - Hetzner: Improved autoscaler and deployment error reporting
  - Improved BYOK location handling and error reporting
  - Added support for AWS ephemeral disks with any device identifier

- **Workloads**
  - Enabled Capacity AI in-place pod resizing for stateful workloads
  - Added sticky session support for service-to-service traffic within the same GVC, including port-level configuration
  - Enhanced domain mirroring to support routing to workloads in other GVCs
  - Added validation to reject invalid cron schedule expressions
  - Fixed a case where OOM errors were not surfaced in workload events
  - Fixed serverless workloads scaled to 0 showing incorrect readiness status

- **Volumes**
  - Volume expansion rate limited to 4 expansions per 24-hour period
  - Fixed reactive volume scaling triggering prematurely

- **Domains & DNS**
  - Allow domains with CNAME and subdomain routing when using dns01
  - Improved certificate change detection for provider, DNS names, and certificate name changes
  - Added validation to disallow CNAME records under NS domain parents

- **Billing**
  - Added invoice generation for accounts paid through AWS Marketplace
  - Organizations can now be moved to AWS Marketplace-paid accounts

- **CLI**
  - Released CLI version 3.10.0
  - Added `--identity` flag to `workload cron run` for identity-scoped runner workloads
  - Added `--active-deadline-seconds` option to cron start
  - Added `cpln volumeset shrink` command
  - Added predictive autoscaling support to `volumeset create` and `volumeset update`
  - Fixed ConfigMap and Secret name collision in K8s Converter

- **Terraform Exporter**
  - Added OpenTelemetry logging export support
  - Added predictive scaling support for volume sets
  - Added domain route mirroring support
  - Added workload schedule support
