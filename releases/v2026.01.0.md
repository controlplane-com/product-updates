# Release 2026.01.0

## Platform Updates - January 2026

- **Managed Kubernetes**
  - Added k3s version 1.34.2 with improved stability
  - AWS: Improved nodepool refresh handling for credential updates
  - GCP: Enhanced autoscaler behavior to respect manual scaling adjustments and target size updates
  - Multi-zone snapshot restoration support
  - New optional setting to suppress default storage class creation for BYOK clusters
  - Added Hetzner server types
  - Improved BYOK addon configuration with stricter validation
  - Enhanced storage replicas configuration handling for Generic clusters

- **Workloads**
  - Added fallback KEDA configuration for improved autoscaling reliability
  - Enhanced autoscaling error messaging and configuration validation
  - Fixed CPU thin provisioning for workloads with decimal values
  - Improved serverless replica status accuracy
  - Enhanced replica direct TCP routing with port-based decisions
  - Fixed stale deployment issues when using metric-based autoscaling
  - Improved KEDA authentication error messages

- **Billing**
  - Enhanced billing dashboard with improved user interface
  - Simplified monthly invoice generation process
  - Added GVC search capability in billing interface
  - Improved outstanding balance display

- **Domains & DNS**
  - Improved DNS CNAME configuration display for subdomains
  - Enhanced DNS validation messaging
  - Added support for chained CNAME records when using dns01 certificate verification

- **API Documentation**
  - Introduced "unmodifiable" property for resource protection
  - Improved documentation navigation and clarity

- **Monitoring & Observability**
  - Added OpenTelemetry log forwarding support
  - Enhanced metrics collection and querying capabilities

- **Networking**
  - Improved wormhole tunnel reconnection logic

- **Security & Reliability**
  - Enhanced volume operation conflict resolution
  - Fixed connection pooling issue affecting autoscaling
  - Better handling of expiring certificates for misconfigured workloads
