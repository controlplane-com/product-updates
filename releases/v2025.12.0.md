# Release 2025.12.0

## Platform Updates - December 2025

- **Managed Kubernetes**
  - Added k3s version 1.34.2
  - GCP: Improved autoscaler behavior to respect manual scaling adjustments
  - AWS: Improved node pool refresh handling for credential updates
  - Added a new optional setting to suppress default storage class creation for BYOK clusters
  - Improved autoscaler default values
  - Added GCP provider support to terraform-exporter

- **Workloads**
  - Added fallback KEDA configuration for improved autoscaling reliability
  - Improved error messaging for autoscaling configuration issues
  - Fixed issue with stale deployments when using metric-based filtering
  - Automatic certificate provisioning on org creation for BYOK clusters

- **Billing**
  - New billing-admin dashboard with improved user interface
  - Simplified monthly invoice generation process
  - Added search capability for GVCs in billing interface
  - Improved outstanding balance display
  - Enhanced billing metrics and reporting

- **Domains & DNS**
  - Show correct ACME alias for subdomain certificates
  - Improved DNS validation messaging for configured domains

- **API Documentation**
  - Introduced resource protection with "unmodifiable" property
  - Improved API documentation clarity and navigation

- **Networking**
  - Improved wormhole tunnel reliability with automatic recovery
