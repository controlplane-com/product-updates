# Release 2026.02.0

## Platform Updates - February 2026

- **Managed Kubernetes**
  - AWS: Added nested virtualization support for nodepools
  - AWS: Added KMS key support for node encryption
  - AWS: Added support for bare metal instance types
  - Upgraded Calico networking
  - Adjusted CoreDNS resource configuration for better reliability

- **Workloads**
  - Added in-place pod resizing for CPU and memory changes without restart for standard workloads
  - Added domain mirroring to route requests to other workloads in the same GVC
  - Added publishNotReadyAddresses support via workload tags
  - Fixed potential networking issues for suspended cron workloads
  - Improved KEDA autoscaling trigger validation

- **Volumes**
  - Added volume shrinking capability for reducing volume size
  - Added predictive volume set scaling
  - Improved volume deletion handling for multi-location deployments
  - Allow concurrent `expandVolume` and `createVolumeSnapshot` operations

- **Domains & DNS**
  - Added support for `_verify` subdomain for simplified domain verification

- **CLI**
  - Added `cpln audit` command to retrieve audit trail events
  - Added Domain conversion support to K8s Converter
  - Improved shell completion for file paths and option values
  - Added pagination warnings when additional records are available
  - Enhanced `workload run` capability with dedicated `cron run` subcommand
  - Added org and standard options to agent command

- **Monitoring & Observability**
  - Added CRI-O container runtime support for metrics collection
